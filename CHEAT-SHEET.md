# Final Review Cheat Sheet

## 1. System Frame
- Cardiovascular system = heart-lung pumping unit + peripheral circulation
- Normal-physiology assumptions:
  - pulmonary arterial pressure approximately constant
  - RV in normal operating range
  - LV not failing
- Normal physiology = Region 1

## 2. Most Important Graphs

### Steady State
- Steady state occurs where:
  - cardiac output curve
  - venous return curve
  - intersect

### Venous Return Curve
- `VR = (PMS - Pra) / RVR`
- PMS = x-intercept
- resistance changes slope
- low Pra -> venous collapse -> plateau

### Cardiac Output Curve
- slope depends on HR and contractility
- flattens at high filling pressure

### High-Yield Shifts
- blood volume up -> VR right
- hemorrhage -> VR left
- resistance up -> VR flatter
- AV fistula -> VR steeper
- HR or contractility up -> CO up

## 3. PV Loop Moves
- `SV = EDV - ESV`
- `CO = HR x SV`
- `EF = SV / EDV`
- preload up -> loop right, SV up
- afterload up -> loop taller/narrower, ESV up
- contractility up -> ESPVR steeper, ESV down, SV up

## 4. ECG Core Idea
- ECG = projection of a moving net dipole
- different leads = different views of same event
- limb leads are not independent
- chest leads add out-of-plane information
- V1 P wave normally biphasic

## 5. Rhythms Explicitly Called Out in Final Review

### AF
- irregularly irregular
- no P waves
- irregular AV node bombardment

### Flutter
- regular
- sawtooth inverted waves in inferior leads
- atrial rate about 300
- 150 bpm can be 2:1 flutter

### VT
- rate > 100
- wide QRS
- P waves may be absent or dissociated

### Heart Blocks
- first-degree: long PR, all conduct
- second-degree: some conduct, some do not
- third-degree: atria and ventricles independent

## 6. Equations
- `Q = ΔP / R`
- `CO = HR x SV`
- `SV = EDV - ESV`
- `EF = SV / EDV`
- `CO = VO2 / (CaO2 - CvO2)`
- `VR = (PMS - Pra) / RVR`

## 7. Last-Minute Traps
- most exam questions reduce to curve shifts or lead projections
- regular rhythm near 150 -> think flutter
- wide fast rhythm -> think VT
- irregularly irregular with no P waves -> think AF
