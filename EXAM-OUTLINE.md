# Final Review Exam Outline

## 1. Main Frame: Heart-Lung Pumping Unit and Peripheral Circulation

### System Organization
- The cardiovascular system is usefully divided into:
  - the heart-lung pumping unit
  - the peripheral circulation
- Blood returns to the right heart, passes through the lungs, reaches the left heart, and then enters the peripheral circulation

### Flow Path Review
- RA -> tricuspid valve -> RV
- RV -> pulmonic valve -> pulmonary artery -> lungs
- Pulmonary veins -> LA
- LA -> mitral valve -> LV
- LV -> aorta -> peripheral circulation

### Right Ventricle as the Normal Limiting Pump
- In normal physiology, cardiac output is mainly determined by right ventricular performance
- Review-session idea:
  - RV is the weaker limiting pump
  - LV is the stronger downstream pump

### Assumptions for the Simplified Normal Physiology Model
- Pulmonary arterial pressure is approximately constant
- The right ventricle is in its normal operating range
- The left ventricle is not failing
- These assumptions define the normal regime emphasized in final review

## 2. Cardiac Output / Venous Return Graphical Reasoning

### Why This Matters
- The review session emphasized that if you understand these curves, you can answer most exam questions
- Steady state occurs where cardiac output equals venous return

### Key Equations
- `Q = ΔP / R`
- `CO = HR x SV`
- `VR = (PMS - Pra) / RVR`
- `CO = VO2 / (CaO2 - CvO2)`

### Venous Return Curve
- x-intercept = PMS
- Slope depends on resistance to venous return
- At very low right atrial pressure, venous collapse causes a plateau

### Cardiac Output Curve
- Slope depends on heart rate and contractile state
- The curve flattens at high filling pressures
- Resistance affects steady state through the venous return side

### High-Yield Curve Changes
- Blood volume up -> PMS up -> VR curve shifts right
- Hemorrhage -> PMS down -> VR curve shifts left
- Resistance up -> VR slope becomes flatter
- AV fistula -> resistance down -> VR slope becomes steeper
- Heart rate up -> CO curve increases
- Contractility up -> CO curve increases

### Three Regions
- Region 1: normal physiology, RV-limited operating range
- Region 2: RV filling/volume limit reached
- Region 3: LV failure limit
- The review emphasized that normal physiology questions are mainly Region 1 questions

## 3. Pressure-Volume Loop Reasoning

### Core Cardiac-Cycle Phases
- Filling
- Isovolumetric contraction
- Ejection
- Relaxation

### Key Relationships
- `SV = EDV - ESV`
- `CO = HR x SV`
- `EF = SV / EDV`

### PV Loop Logic
- Increased preload -> loop shifts right, SV rises
- Increased afterload -> loop gets taller and narrower, ESV rises
- Increased contractility -> ESPVR steepens, ESV falls, SV rises

### Main Interpretation Goal
- Use the loop to infer preload, afterload, contractility, and stroke volume

## 4. ECG Theory from the Final Review

### Single Equivalent Moving Dipole
- The ECG is the projection of a time-dependent net cardiac dipole
- Different leads are different projections of the same underlying electrical event
- This was the main conceptual ECG framework emphasized in final review

### Limb Leads
- Lead I = LA - RA
- Lead II = LL - RA
- Lead III = LL - LA
- The frontal-plane limb leads are not mathematically independent

### Chest Leads
- Chest leads provide information outside the limb-lead plane
- They add out-of-plane spatial information that limb leads cannot provide alone

### High-Yield Waveform Logic
- P wave = atrial depolarization
- QRS = ventricular depolarization
- T wave = ventricular repolarization
- V1 P wave is normally biphasic
- The T wave is often in the same general direction as the dominant QRS

### Axis Logic
- Find the most isoelectric limb lead
- The mean axis is perpendicular to it
- Use polarity of neighboring leads to choose the correct direction

## 5. Rhythm Recognition Explicitly Prioritized in Review

### Atrial Fibrillation
- Irregularly irregular rhythm
- No visible P waves
- Atrial myocytes are depolarizing independently
- The AV node is bombarded irregularly
- Ventricular rhythm is therefore random
- Usually tachycardic

### Atrial Flutter
- Typically regular
- Concerted atrial activity rather than chaotic activity
- Inverted sawtooth flutter waves in inferior leads
- Atrial rate about 300/min
- Ventricular rate often `300 / n`
- Classic test pattern: 150 bpm can be 2:1 flutter

### Ventricular Tachycardia
- Ventricular rate > 100
- QRS is typically wide
- P waves may be absent
- If present, P waves may be unrelated to the QRS
- AV dissociation is a key concept

### Heart Blocks
- First-degree: PR > 200 ms, every P wave conducts
- Second-degree: some P waves conduct, some do not
- Third-degree: atrial and ventricular rhythms are independent
