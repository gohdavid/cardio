# High-Yield Must Know From Final Review

## The 4 Big Tested Buckets
- CO / venous return graphical reasoning
- PV loop reasoning
- ECG dipole / lead projection logic
- Recognition of AF, flutter, VT, and heart blocks

## Core Assumptions
- pulmonary arterial pressure approximately constant
- RV in normal operating range
- LV not failing
- normal physiology = Region 1

## Core Curve Logic
- steady state = venous return = cardiac output
- PMS sets the venous return x-intercept
- resistance changes the venous return slope
- HR and contractility change the cardiac output curve

## Core PV Logic
- `SV = EDV - ESV`
- `CO = HR x SV`
- preload up -> SV up
- afterload up -> ESV up, SV down
- contractility up -> ESV down, SV up

## Core ECG Logic
- ECG is the projection of a moving net dipole
- different leads = different views of the same electrical event
- limb leads are not independent
- chest leads add out-of-plane information
- V1 P wave is normally biphasic

## Must-Recognize Rhythms
- AF: irregularly irregular, no P waves
- Flutter: regular, sawtooth, atrial rate about 300
- VT: wide, fast ventricular rhythm, may have AV dissociation
- First-degree block: long PR, every P conducts
- Second-degree block: some P waves conduct, some do not
- Third-degree block: P waves and QRS are independent

## Hallmark Traps
- 150 bpm can be atrial flutter with 2:1 block
- wide tachycardia should make you think VT
- AF is random because the AV node is bombarded irregularly

## Equations
- `Q = ΔP / R`
- `CO = HR x SV`
- `SV = EDV - ESV`
- `EF = SV / EDV`
- `CO = VO2 / (CaO2 - CvO2)`
- `VR = (PMS - Pra) / RVR`
