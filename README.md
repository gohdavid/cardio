# HST.090 / HST.091 — Cardiovascular Physiology & Electrophysiology
## Lecture Notes — Spring 2026

## Course Overview

This series covers the fundamental cardiovascular physiology and clinical electrophysiology taught in the HST.090/HST.091 combined curriculum. Topics span from basic cardiac anatomy and hemodynamics through clinical EKG reading, arrhythmia management, autonomic control, and vascular physiology. Clinical EKG lectures were delivered by Dr. Tatro (Electrophysiologist); physiology lectures by the course faculty.

## Key Equations Reference

| Equation | Description |
|----------|-------------|
| T = Pr/2h | Laplace's Law: wall tension |
| Q = ΔP/R | Ohm's Law for blood flow |
| CO = HR × SV | Cardiac output |
| CO = VO₂ / (CaO₂ – CvO₂) | Fick principle |
| EF = SV / EDV | Ejection fraction |
| PMS = stressed volume / total compliance | Mean systemic filling pressure |
| VR = (PMS – Pf) / R_total | Venous return |
| PP ≈ SV / Ca | Pulse pressure |
| E_ion = (RT/zF) ln([ion]o/[ion]i) | Nernst equation |
| φ = (m · cosθ) / (4πσr²) | ECG surface potential from dipole |
| QTc = QT / √(RR) | Corrected QT interval (Bazett) |
